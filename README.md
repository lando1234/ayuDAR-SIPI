# ayuDAR-SIPI

Para poder correrlo:

- Tener instalado node.
- Clonar el repo.
- Abrir la terminal
- ir a {path del repo}/ayudar
- Correr ```npm install```
- Una vez que termina de correr ejecutar ```npm run dev``` para poder ver los cambios que vaya haciendo.
- La aplicación corre en localhost:3000
